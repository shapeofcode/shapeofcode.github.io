1. worker node lable 확인

```shell
kubectl get node --show-labels
kubectl get node -L disktype

kubectl run eshop-store --image=nginx --dry-run=client -o yaml > eshop-store.yaml

vi eshop-store.yaml

apiVersion: v1
kind: Pod
metadata:
   name: eshop-store
spec:
   nodeSelector:
      disktype: ssd
      ...
```
