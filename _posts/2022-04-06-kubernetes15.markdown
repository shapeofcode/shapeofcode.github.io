1.namespace 생성

kubectl create namespace <namespace-name>

<br/>

2.pod 생성

kubectl run <pod-name> --image=<image-name> --env=<env> --dry-run=client --namespace=<namespace-name> -o yaml > pod.yaml

<br/>

3.확인

kubectl get pod -n <namespace-name>
